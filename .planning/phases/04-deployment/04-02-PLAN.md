<task type="checkpoint:human-action" gate="blocking">
  <decision>Deploy to Railway</decision>
  <context>
    The codebase is configured for Railway deployment. Now deploy using Railway CLI or dashboard.

    **Option A: Railway CLI (Recommended)**
    ```bash
    # Install Railway CLI if not installed
    npm install -g @railway/cli

    # Login to Railway
    railway login

    # Initialize project (in project directory)
    railway init

    # Add environment variables
    railway variables set OPENROUTER_API_KEY=your_key_here
    railway variables set SUPABASE_URL=your_supabase_url
    railway variables set SUPABASE_ANON_KEY=your_anon_key
    railway variables set VITE_API_URL=""

    # Deploy
    railway up
    ```

    **Option B: Railway Dashboard**
    1. Go to https://railway.app
    2. Click "New Project" -> "Deploy from GitHub repo"
    3. Select this repository
    4. Add environment variables in the Variables tab
    5. Railway will auto-deploy

    **Required Environment Variables:**
    | Variable | Source |
    |----------|--------|
    | OPENROUTER_API_KEY | OpenRouter Dashboard -> Keys |
    | SUPABASE_URL | Supabase Dashboard -> Project Settings -> API |
    | SUPABASE_ANON_KEY | Supabase Dashboard -> Project Settings -> API |
    | VITE_API_URL | Set to empty string "" |
  </context>
